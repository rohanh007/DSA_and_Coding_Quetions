# Graph Data Structure
####  " Graph is the type of data structure which combination of nodes and edges "

"A graph is like a social network. You have people (nodes) and their relationships (edges). Each person is a node, and the connections between people represent friendships or interactions. For example, if you have a group of friends, each friend is a nodes, and the friendships between them are the edges. Graphs are used to represent all sorts of relationships, from social networks to road maps. "
